3. Organizing data

4 points possible

  • 2pts - fix data
  • 2pts - describe problems

The original data file had duplicated column names (length and diameter), relying on another column to differentiate the anatomical element. This is a no, no. Column names must stand alone (e.g.Â tibia_diameter).

The rows containing species names violate the 1 row = 1 observation priciple. These rows should be turned into a column which records the species for each observation.
