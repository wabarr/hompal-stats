Tips On Homework


1. Put your name in the file-name

Something like barr_hw1.html. Always zip the file!




2. Always diplay output!

If you are answering a question, don’t make me study your code to know if it worked….be sure to show me.

Don’t do this….

myVar <- rnorm(100)
negatives <- myVar[myVar<0]

Do this…

myVar <- rnorm(100)
negatives <- myVar[myVar<0]
