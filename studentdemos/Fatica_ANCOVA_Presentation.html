For this demonstartion, we will be using the dataset from Boddy et al. 2012 that examines the allometric relationship between brain and body size. Specifically, we will be testing the hypothesis that Primates and Cetartiodactyla have the same allometric relationship between log body size and log brain size.

First, we need to read in the data, log transofrm brain size and body size, and restrict our data frame to just the data in which we are interested.

In order to visualize the data, we must subset it further into the groups we are interested in: Cetartiodactyla, Primates, and all other mammal orders.

cetartiodactyla <- filter(all.mammals, Order=="Cetartiodactyla")
primates <- filter(all.mammals, Order=="Primates")
other.mammals <- filter(all.mammals, !Order=="Cetartiodactyla", !Order=="Primates")

Now, we will create a scatter plot that represnts Primates with red points, Cetartiodactyla with blue points, and all other mammals with gray points.

library(ggplot2)

cetartiodactyla.lm <- lm(data=cetartiodactyla, logBrain.Mass~logBody.Mass)

We can see that the slopes for Primates, Cetartiodactyla, and the other mammals are not the same. We use ANCOVA to test whether this visual difference is significant.

In order to perform the ANCOVA, we will first add two columns to the all.mammals dataframe: one that separates order Primates from the rest of the mammals and one that separates order Cetartiodactyla from the other mammals.

brains$is.primate <- brains$Order=="Primates"
brains$is.cetartiodactyla <- brains$Order=="Cetartiodactyla"

Finally, we will perform an ANCOVA for both Primates and Cetartiodactyla. We will use the slope.com() function that is part of the package “smatr”. Note that this package is using reduced/standardized major axis regression to perform the ANOVA.

When we use OLS, Primates is significantly different from all othermammals, but Cetartiodactyla is not. This highlights the differences between OLS and RMA/SMA. In this case, since there is error in both the x and y measurements, the RMA/SMA regression is more appropriate.
